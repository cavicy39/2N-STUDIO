# 2N-STUDIO - Web Development Company

A modern, responsive website for 2N-STUDIO, a professional web development company specializing in creating high-performance websites and web applications.

## Features

- **Responsive Design**: Mobile-first approach that works seamlessly on all devices
- **Modern UI/UX**: Clean, professional design with smooth animations
- **Performance Optimized**: Fast loading times with optimized code
- **Interactive Elements**: Smooth scrolling, mobile menu, form validation
- **Professional Sections**:
  - Hero section with call-to-action
  - About section with company stats
  - Services showcase
  - Portfolio gallery
  - Contact form
  - Footer with quick links

## Technologies Used

- HTML5
- CSS3 (with CSS Variables and Flexbox/Grid)
- Vanilla JavaScript (ES6+)
- Google Fonts (Inter)

## Getting Started

Simply open `index.html` in your web browser to view the website.

```bash
# Clone the repository
git clone https://github.com/cavicy39/2N-STUDIO.git

# Navigate to the directory
cd 2N-STUDIO

# Open in browser
open index.html
```

Or use a local server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx http-server

# Then visit http://localhost:8000
```
